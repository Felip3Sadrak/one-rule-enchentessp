## Sobre o Projeto

O **One Rule EnchentesSP** é um projeto de modelagem preditiva em Python que utiliza o algoritmo **One Rule (Uma Regra)** para prever o risco de alagamentos em zonas periféricas da cidade de São Paulo. O modelo considera variáveis críticas, como **tipo de solo**, **topografia** e **infraestrutura de drenagem**, para identificar áreas vulneráveis e auxiliar no planejamento urbano e na prevenção de enchentes.

## Questões Abordadas

O projeto busca responder às seguintes perguntas:

1. **Infraestrutura de Drenagem**: Como a qualidade da drenagem afeta a incidência de alagamentos?
2. **Tipo de Solo**: De que forma o tipo de solo influencia a absorção de água?
3. **Topografia**: Como a topografia contribui para o acúmulo de água?
4. **Impactos Sociais e Econômicos**: Quais são as consequências dos alagamentos para as comunidades?

## Objetivos

- **Identificar Fatores Críticos**: Avaliar a influência da topografia, tipo de solo e infraestrutura de drenagem no risco de alagamentos.
- **Modelagem Preditiva**: Utilizar técnicas de Machine Learning para prever a probabilidade de alagamentos e identificar áreas vulneráveis.
- **Contribuição para Planejamento Urbano**: Fornecer informações para melhorar a infraestrutura de drenagem e a gestão de recursos hídricos.

## Tecnologias Utilizadas

- **Python 3.x**
- **Scikit-learn**: Modelagem preditiva utilizando o algoritmo One Rule.
- **Pandas**: Manipulação e análise de dados.
- **NumPy**: Operações matemáticas.
- **Matplotlib / Seaborn**: Visualização de dados.
- **collections.Counter**: Contagem de classes para análise de frequências.
