## Sobre o Projeto

O **One Rule EnchentesSP** é um script desenvolvido em Python que utiliza o método de aprendizado de máquina **One Rule (Uma Regra)** para prever o risco de enchentes em zonas periféricas da cidade de São Paulo. O script realiza uma análise baseada em variáveis como **tipo de terreno**, **região**, **nível de chuva** e **nível do rio**, proporcionando uma ferramenta simples e eficiente para auxiliar na tomada de decisões preventivas contra enchentes.

### Objetivos

- Prever a probabilidade de risco de enchentes em áreas vulneráveis.
- Analisar a influência do tipo de terreno e da região no risco de enchentes.
- Facilitar a análise para órgãos de monitoramento e defesa civil.

## Tecnologias Utilizadas

- **Python 3.x**
- **Pandas**: Manipulação e análise de dados.
- **Scikit-learn**: Implementação do método One Rule.
- **NumPy**: Operações matemáticas e manipulação de arrays.
- **Matplotlib**: Visualização de dados (opcional para gráficos).
- **Jupyter Notebook**: Para testes e desenvolvimento iterativo (opcional).
